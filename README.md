# مشواري (Mashwari) - Ride Hailing App

A clean, modern, mobile-first ride-hailing application built with React, Tailwind CSS, and Framer Motion.

## Features
- **Mobile-First Design**: Optimized for mobile screens with a native app feel.
- **RTL Support**: Full Arabic language support with Right-to-Left layout.
- **Rider & Driver Modes**: Switch between roles to experience both sides of the app.
- **Real-time Simulation**: Smooth transitions and state management for ride requests.

## Tech Stack
- **React 19**
- **Tailwind CSS 4**
- **Motion (Framer Motion)**: For smooth animations and layout transitions.
- **Lucide React**: For clean, modern icons.

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the app in your browser (usually at `http://localhost:3000`).

## Usage
- **Rider Mode**: Click "طلب مشوار" (Request Ride) to create a new request.
- **Driver Mode**: Go to the "Requests" tab to see pending rides and click "قبول الطلب" (Accept Ride).
- **Toggle Role**: Use the profile/car icon in the bottom navigation to switch between Rider and Driver views.
